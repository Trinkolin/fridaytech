Les fixtures sont des fichiers externes contenant des données que les tests Cypress peuvent utiliser. Ces données peuvent être de n'importe quel type, y compris des chaînes de caractères, des tableaux, des objets, des images, des fichiers JSON, etc.

Les fixtures sont stockés par défaut dans le dossier:

cypress/fixtures

Pour utiliser un fixture, vous pouvez utiliser la commande:

cy.fixture()

Les fixtures peuvent être utilisées dans une grande variété de cas d'utilisation, comme pour :

  • Fournir des données pour les tests de validation
  • Fournir des données pour les tests de performances
  • Simuler des requêtes réseau
  • Générer des données aléatoires
